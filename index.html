Hello, I'm here to say the git commands that I have learnt which are:

  • git clone - clone repositories into your local computer.
  • git push - push changes to the main branch.
  • pwd - shows which is you present working directory.
  • cd - used to change working directory.
  • git commit - commits all changes in that file.
  • git pull - pull changes to working directory.
  • git add - to update what will be committed.
  • git restore - to discard changes in working directory.
